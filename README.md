 BÀI THỰC HÀNH SỐ 1 – MICROSOFT WORD (MAIL MERGE)
 Những việc bạn cần làm:

Tạo file Excel (dữ liệu nguồn)

Nhập danh sách gồm các cột:

Title

First Name

Surname

Address

City

Country

Lưu file Excel để dùng cho trộn thư.

Tạo file Word (thư mẫu)

Soạn nội dung thư theo mẫu cho sẵn.

Chèn các trường trộn thư (Mail Merge Fields) tương ứng với dữ liệu trong Excel.

Định dạng văn bản rõ ràng, đúng bố cục.

Thực hiện Mail Merge

Kết nối file Word với file Excel.

Trộn thư để tạo nhiều thư cho từng người trong danh sách.

Lưu kết quả

Lưu thư mẫu.

Lưu file Word sau khi trộn thư hoàn chỉnh.

 Mục tiêu bài 1:
Biết sử dụng Mail Merge để tạo văn bản hàng loạt từ dữ liệu Excel.

BÀI THỰC HÀNH SỐ 2 – MICROSOFT WORD (MỤC LỤC)
Những việc bạn cần làm:

Nhập nội dung văn bản nhiều trang

Gồm các phần: Giới thiệu, lý do chọn đề tài, phạm vi nghiên cứu, nội dung nghiên cứu, cơ sở lý thuyết, các chương/mục.

Định dạng Heading

Heading 1: Chương

Heading 2: Mục

Heading 3: Tiểu mục
→ Bắt buộc dùng Heading để Word nhận diện khi tạo mục lục.

Tạo mục lục tự động

Đặt con trỏ tại trang Mục lục.

Insert → Table of Contents.

Cập nhật mục lục

Khi chỉnh sửa nội dung, dùng Update Table để cập nhật lại số trang.

Mục tiêu bài 2:
Biết tạo và quản lý mục lục tự động trong Word.

BÀI THỰC HÀNH SỐ 3 – GOOGLE DOCS
3.1 Tạo file & chia sẻ

Tạo file Google Docs mới.

Đặt tên: Buoi2_Họ_tên_Bai_1.gdoc

Chia sẻ:

Anyone with the link (Viewer).

3.2 Định dạng Page Setup, Font, Paragraph

Page Setup (File → Page setup):

Top: 1 inch

Bottom: 1 inch

Left: 1.2 inch

Right: 0.8 inch

Orientation: Portrait

Paper: A4

Font:

Times New Roman

Size: 13

Paragraph:

Before: 6 pt

After: 6 pt

Line spacing: 1.5

3.3 Tạo và định dạng văn bản

Gõ và trình bày ĐƠN XIN XÁC NHẬN theo mẫu:

Quốc hiệu, tiêu ngữ (canh giữa, có gạch ngang).

Tên đơn in hoa, đậm.

Phần kính gửi.

Thông tin cá nhân (theo mẫu chấm).

Nội dung đơn xin xác nhận là sinh viên.

Ngày tháng, chữ ký người làm đơn.

Phần xác nhận của nhà trường.

3.4 Chia sẻ quyền – History – Meeting

Đặt General access: Restricted.

Chia sẻ file cho 3 người:

1 Editor

1 Viewer

1 Commenter

Dùng Comment/Suggest để thảo luận chỉnh sửa nội dung.

Xem Version history để kiểm tra lịch sử chỉnh sửa.

Mục tiêu bài 3:

Biết sử dụng Google Docs, định dạng văn bản, chia sẻ quyền, comment và kiểm tra lịch sử chỉnh sửa.
